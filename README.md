## Topics Covered

### Fundamentals
- **Variables**: `let`, `const`
- **Data Types**: Numbers, Strings, Booleans, Arrays, Objects
- **Operators**: Comparison (`===` vs `==`), Ternary operators
- **Control Flow**: `if/else`, `switch`, loops (`for` & `for...of`)

### Functions
- Function declarations vs expressions
- Arrow functions (ES6)
- Higher-order functions and callbacks
- Default parameters
- Rest parameters (`...`)

### More Concepts
- Template literals
- Error handling (`try/catch/finally`)
- Date manipulation
- String methods
- JSON parsing
